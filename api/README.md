# BackEnd-nwoork

Esta Aplicacion fue realizada en NestJs

## Requisitos
```sh
    Node -v > 14.8
    Postgresql
```

Primeros pasos
```sh
npm install --force
```

Comandos para ejecutar:

```sh
npm run dev
```

Variables de entorno:
```sh
PORT=3000
DB_USER='user'
DB_PASSWORD='password'
DB_HOST='localhost'
DB_NAME='task'
DB_PORT=5432
```
